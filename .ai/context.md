## Key Constraints

Hard rules the AI must not violate. Be specific.

- **No new top-level dependencies** without updating `.ai/decisions.md`.
- **No breaking changes to the public API** in `src/api/v1/` — version under `v2/` instead.
- **All DB access goes through the repository layer** in `src/repositories/`.
  Do not call the ORM directly from routes or services.
- **Secrets only via env vars**, never committed. `.env.example` lists required keys.
- **Python is 3.12+** — `match` statements and PEP 695 generics are allowed.
- **Target runtime is Linux x86_64.** Do not add macOS-only or Windows-only code paths.
- **Every new endpoint needs at least one integration test** in `tests/integration/`.

## Entry Points

Where to start reading when picking up cold.

- **Run the app:** `make dev` (or `uvicorn src.main:app --reload`)
- **Run tests:** `make test` (or `pytest -xvs`)
- **Main entry file:** `src/main.py` — wires FastAPI app, middleware, routers.
- **HTTP routes:** `src/api/v1/` — one file per resource.
- **Business logic:** `src/services/` — framework-agnostic.
- **Data access:** `src/repositories/` — only layer that touches the DB.
- **Config:** `src/config.py` — reads from env, validates with Pydantic.
- **Migrations:** `migrations/versions/` — generated by Alembic.
